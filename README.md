{Deprecated} Bitbucket Server API wrapper for .Net Core
============================================================================

**This project is deprecated** 

Please use https://github.com/jlouros/StashApiCSharp instead. Since all methods provided this wrapper are supported by [BitBucket Server](https://www.atlassian.com/software/bitbucket/server) and [Stash](https://confluence.atlassian.com/bitbucketserver/bitbucket-rebrand-faq-779298912.html#BitbucketrebrandFAQ-WhydidwerebrandStashtoBitbucketServer?) there is not need to have both projects separated.
